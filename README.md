# Multiplayer FPS Framework (Source Code)

**Role:** Systems Engineer / Solo Developer
**Engine:** Unreal Engine 5.4
**Language:** C++

## Overview
A modular First-Person Shooter framework engineered to serve as a scalable foundation for competitive multiplayer games.
This project integrates Epic Online Services (EOS) for matchmaking and user identity management.

**Note:** This repository contains **source code only**.

## Key Technical Features
* **EOS Integration:** Peer-to-peer matchmaking and user authentication using Epic Online Services.
* **Lag Compensation:** Implemented client-side prediction for weapon firing and movement to ensure responsive gameplay in high-latency environments.
* **Replication Architecture:** Scalable class hierarchy for weapons and a replicated animation state machine designed for bandwidth efficiency.
* **Player System:** Scalable class for modular player systems.
* **Weapon System:** Scalable class for weapon sytems and types
* **Attachment Component System:** Scalable class for modular weapon attachments.
* **Abilities System:** Scalable class for modular player abilties.
